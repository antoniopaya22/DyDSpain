# Llama permanente

**Evocación — Nivel 2**

---

- **Tiempo de lanzamiento:** 1 acción
- **Alcance:** Toque
- **Componentes:** V, S, M (polvo de rubí con un valor
- **Duración:** Hasta que sea disipado

---

Una llama, con un brillo igual al de una antorcha,
brota de un objeto que tocas. El efecto es el de una
llama normal, pero no genera calor ni consume
oxígeno. Una _llama permanente_ se puede tapar
u ocultar, pero no se puede apagar ni extinguir.
